WooCommerce Sales Order Audit Log plugin helps in tracking changes made to Sales Order 

== Description ==
Lite version tracks changes like Order Item Edit, New Item Add, Move to Trash, Restore from Trash.

**Features in Free Version**:
* Order Item Edit (Display edited Product's Name).
* Order New Item Added.
* Order Move To Trash.
* Restore Order from Trash.

**Pro version tracks following additional changes made to Sale Order**:
* Order Deleted.
* Order Item Deleted.
* Order Item Edited.
* New Item Added.
* Order Status Change.
* Order Trash.
* Restore Order.
* Refund Add.
* Refund delete.
* Notes delete.
* Edited Date and Customer Name.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for 'Order Audit Log for WooCommerce').
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/order-audit-log.zip) the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Access the settings page from WordPress backend under WooAudit Lite
